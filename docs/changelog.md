# История изменений и идеи

## Что сделано

- Настроен GitHub репозиторий и GitHub Pages
- Настроен Supabase: таблицы, RLS, Realtime, GRANT права
- Авторизация через токен в URL (без email, без формы входа)
- Scoreboard с Realtime подпиской (WebSocket, без polling)
- Страница судьи: модал участников, таблица ввода, рейтинг внизу
- Логика пар: щит 1+2, 3+4 (не два стрелка на одном щите)
- Заголовки серий: 5 5 5 3 3 3 (количество выстрелов в серии)
- Enter → переход вниз по строкам; Tab → следующий столбец
- Рейтинг на странице судьи без дублирования данных

## Идеи для следующей версии

- Индикатор изменения позиции (↑ ↓)
- Поддержка произвольного количества пар
- История соревнований
- QR-код для зрителей
- Полноэкранный режим для scoreboard
