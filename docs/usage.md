# Руководство пользователя

## Перед каждым соревнованием

1. Открыть ссылку судьи (сохранена в телефоне/мессенджере)
2. Нажать **+ Новое соревнование**
3. Ввести фамилии 8 стрелков (4 пары, щиты 1–8), Enter — следующая строка
4. Нажать **Старт →**
5. Убедиться что scoreboard открыт на экране для зрителей

> Это может сделать любой у кого есть ссылка судьи — не только владелец аккаунта.

---

## Во время соревнования (судья)

1. Открыть сохранённую ссылку судьи
2. После каждой серии — ввести сумму серии в ячейку (цифра видна на экране установки)
3. Enter — следующая строка, Tab — следующий столбец
4. Всё сохраняется автоматически
5. Рейтинг внизу страницы обновляется сразу — видно кто вылетает

### Навигация в таблице

| Клавиша | Действие |
|---------|---------|
| Enter | ↓ следующая строка (тот же столбец) |
| Tab | → следующий столбец |
| Shift+Tab | ← предыдущий столбец |
| ↑ ↓ | строка выше/ниже |
| ← → | столбец влево/вправо |

---

## Таблица ввода (страница судьи)

- Колонки: Щит | Фамилия | **5 | 5 | 5 | 3 | 3 | 3** | Σ
- Заголовки серий = количество выстрелов в серии
- Всегда 6 серий, автосохранение при уходе из ячейки
- Сортировка не отображается — судья видит стрелков в порядке щитов

---

## Страница зрителей (scoreboard)

- Чёрный фон, белый текст, лидер выделен жёлтым
- Таблица: `#` | Пара | Итого
- Обновляется мгновенно при каждом введённом значении
- При старте нового соревнования страница перезагружается автоматически
- **Авто-реконнект:** при обрыве сети подключение восстанавливается
  автоматически, данные подтягиваются — вручную перезагружать не нужно
  (индикатор в углу: `● live` / `● connecting`)
- **Полноэкранный режим:** двойной клик по странице или клавиша **F**

## Смена токена судьи

Если сменился судья или нужно отозвать доступ:

**Supabase → SQL Editor** → открыть готовый файл `sql/rotate-token.sql`,
заменить `НОВЫЙ_ТОКЕН` на новую строку → **Run**.

Либо вручную:

```sql
CREATE OR REPLACE FUNCTION get_judge_token()
RETURNS text LANGUAGE sql SECURITY DEFINER STABLE AS $$
  SELECT 'НОВЫЙ_ТОКЕН';
$$;
```

Отправить судье новую ссылку:
```
https://sst1xx.github.io/mushketon-final/judge/?token=НОВЫЙ_ТОКЕН
```

Код репозитория при этом **не меняется**.
